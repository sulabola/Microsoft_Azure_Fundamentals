# Microsoft Azure Fundamentals (AZ-900) – Study Notes

This repository contains structured study notes for the **Microsoft Azure Fundamentals (AZ-900)** certification, based on the Udemy course *“Master Microsoft Azure Fundamentals: AZ-900 Exam Prep 2025”* by Scott Duffy.

The notes are organized by course section and cover core Azure concepts, including cloud fundamentals, Azure services, security, governance, pricing, and support. Each section is documented in Word and exported as PDF for easy reference and long-term revision.
